# wialon-js-gmap
Generate Wialon Asset with Google Map and JavaScript

Discription 

Get Wialon assets and display to gmap every x seconds

Usage : 
1. Download and edit file wialon-js-gmap.html
2. Edit token = ""; with your Wialon Token
